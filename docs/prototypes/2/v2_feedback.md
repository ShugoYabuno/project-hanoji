# プロトタイプ v2 試作フィードバック & 次期(v3)検討事項

## 1. 3Dプリント製造上の課題と対策 (2026-01-18)

### 課題: 天板裏面の荒れ
- **現象**: テンティングおよびハの字傾斜により、天板の裏側が広範囲のオーバーハングとなる。`Tree (Default)` サポートで出力した結果、フィラメントの垂れ下がりが発生し、面が荒れてしまった。
- **影響**: ナット締結時の座面が不安定になる恐れがある。

### 対策方針 (次回プリント/v3試作時)

#### アプローチA: 造形方向の変更（推奨）
表面（天板面）の滑らかさを最優先し、積層痕を目立たなくするために**「側面を下にして立てて印刷」**する。

*   **配置 (Orientation)**:
    *   **Y軸合わせ**: モデルの長辺をプリンタのY軸（ベッド移動方向）に平行に合わせる。
        *   *理由*: Bambu Lab A1等のベッド駆動型において、転倒リスクを最小化するため。
*   **定着設定 (Bed Adhesion)**:
    *   **Brim**: `Outer brim only`, 幅 `15mm` 〜 `20mm`。
        *   *理由*: 接地面積が極端に狭いため、ブリムで強制的に足を広げて転倒を防ぐ。
*   **サポート設定**:
    *   **揺れ防止支柱**: `Tree (manual)` を使用し、モデルの中腹（高さの半分〜2/3付近）に手動でサポートポイントを追加して、左右から支える。
        *   *理由*: 高さがあるため、ノズル移動時の振動でモデル上部が揺れ、造形品質が下がるのを防ぐ（リンギング対策）。
*   **印刷速度**:
    *   **Silent Mode**: 必要に応じて速度を50%に落とす。

#### アプローチB: 平置きでの改善（安定重視）
もし立てて印刷して失敗する場合のバックアッププラン。

*   **サポート種類**: `Normal` (Style: `Grid` or `Rectilinear`)
    *   *理由*: Treeよりも土台が安定するため、広い平面を支えるのに適している。
*   **インターフェース設定**:
    *   **Top Z distance**: `0.1mm` 〜 `0.15mm` (通常0.2mmより詰める)
    *   **Top interface spacing**: `0mm` (Dense / 隙間なし)
    *   *理由*: サポート最上面を密な板状にし、本体との隙間を極限まで減らすことで垂れ下がりを物理的に防ぐ。

## 2. 設計・形状へのフィードバック (2026-01-18)

### 構成: 分割構造への回帰
- **現状 (v2)**: 一体成形（シングルプリント）。接合面がなく美しいが、サポート材の出力に時間がかかりすぎる。
- **次期 (v3)**: **「天板」「側面」「底面」の3パーツ構成**にする。
    - **狙い**: プリント時間の短縮と、各パーツの向きを最適化することでサポートを減らし、品質を向上させる。

### 外形形状: 5角形から4角形へ
- **現状 (v2)**: 5角形ベースの形状。
- **次期 (v3)**: **4角形（台形・長方形に近い形状）**に変更する。
    - **狙い**: 手のひらを置くスペース（パームレストエリア）を広く確保するため。

### 面構成: 親指（ジャンプ）エリアの高さ
- **現状 (v2)**: 設計簡易化のため、単純な多面体構成を採用。
- **次期 (v3)**: **ジャンプボタンの面を、パームレスト面よりも一段低く**設定する。
    - **狙い**: 親指の自然な可動域に合わせ、よりリラックスした状態で打鍵できるようにする。

### ボタンレイアウト配置
- **変更**: 移動ボタン群とアクションボタン群を、現在の位置よりも**上方向に約15mm**ずらす。
    - **狙い**: 手のひらの置き位置と指の届き具合の最適化。
