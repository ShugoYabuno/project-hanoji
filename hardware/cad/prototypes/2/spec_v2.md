# プロトタイプ v2 設計仕様書 (Draft)

## 1. 概要
- **目的**: v1の検証結果（保持困難、手首負荷）を解消し、快適な操作性を実現する。
- **主要変更点**:
    - パームレストの一体化（または必須アタッチメント化）
    - XY平面への角度付け（ハの字配置）
    - テンティング角の維持・最適化

## 2. 幾何学パラメータ (Geometry)

### 2.1 角度設定
| パラメータ | 設定値 (暫定) | 備考 |
| :--- | :--- | :--- |
| **テンティング角 (Z軸回転)** | 10° 〜 15° | エルゴノミクス調査に基づく推奨値。v1でも採用。 |
| **ハの字角度 (XY平面回転)** | **30° (片側 15°)** | **確定**。手首の尺側偏位を防ぐため。 |
| **手首の傾斜 (チルト/パームレスト)** | 0° 〜 -5° (ネガティブチルト) | 手首を反らせないための配慮。 |

### 2.2 寸法・配置
- **基本レイアウト**: レバーレス（Hitbox準拠）
- **左右分割**:
    - [ ] 完全分割型
    - [x] **一体型（V字形状）**
        - **理由**: 膝上プレイ時の安定性を最優先するため（格闘ゲーム用途）。
        - **特徴**: 左右のユニットが中央または底面プレートで接続され、固定された「ハの字」と「テンティング」を持つ。
        - **左右間隔 (Split Distance)**: **50mm** (中央部でのボタン群の距離)
- **ボタンサイズ**:
    - アクションボタン: 24mm
    - ジャンプ/親指: 30mm (要確認)

## 3. 構造・コンポーネント
### 3.1 パームレスト
- **要件**: 筐体と一体化、または強固に固定。
- **形状**: 傾斜がついているため、手が滑り落ちないような摩擦係数の高い素材、または「返し」となる形状が必要か検討。
