# プロジェクトHanoji ハードウェア素材選定ガイド

本ドキュメントは、プロジェクトHanojiにおける筐体製造（3Dプリント・外注）の素材選定基準をまとめたものである。
目的（試作、コスト重視、品質重視）に応じて適切な素材・工法を選択すること。

## 1. 素材・コスト比較マトリクス

※コストは1kgフィラメントリールの市場価格目安（外注は部品単価目安）。

| 素材カテゴリ | おすすめ素材 | コスト感 (目安) | 積層痕 | 強度・耐久性 | こんな場合におすすめ |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **デザイン・安価** | **PLA**<br>(Matte / Silk) | **低**<br>(¥3,000〜) | **Matte**: やや隠れる<br>**Silk**: 目立つが綺麗 | **中**<br>熱に弱い (〜55℃) | ・色や光沢で個性を出したい<br>・室内利用のみ<br>・**とにかく安く作りたい** |
| **実用・高コスパ** | **PETG** | **低**<br>(¥3,000〜) | **目立つ**<br>(テカテカする) | **高**<br>割れにくい、薬品に強い | ・**コスパ最強の実用品**を作りたい<br>・持ち運びで乱暴に扱いたい<br>・見た目は二の次 |
| **工業・タフ** | **ASA**<br>(またはABS) | **中**<br>(¥4,000〜) | 普通<br>(後加工で消せる) | **特高**<br>耐熱・耐候性が高い | ・市販品レベルの**頑丈さ**が欲しい<br>・長く使える製品にしたい<br>・アセトン燻蒸等に挑戦したい |
| **高級・質感** | **CF-PLA**<br>**CF-PETG** | **高**<br>(¥5,500〜) | **目立たない**<br>(マットな質感) | **高**<br>カッチリ硬い | ・**自家生産で最高クオリティ**を出したい<br>・塗装なしで高級感を出したい<br>・硬化鋼ノズル等の準備が可能 |
| **究極・プロ** | **Nylon (PA12)**<br>(MJF/SLS外注) | **特高**<br>(¥15,000〜/個) | **なし**<br>(梨地仕上げ) | **最高** | ・**予算度外視**で完璧な製品を作りたい<br>・積層痕を絶対になくしたい<br>・ハイエンドモデルとして販売 |

## 2. 製造コスト試算 (Case Study)

v2プロトタイプ相当（約294g PLA / 一体型 / V字形状）を製造した場合の概算比較。

| 製造方法 | サービス・機器 | 概算コスト (1個あたり) | 特徴 |
| :--- | :--- | :--- | :--- |
| **自家生産 (PLA)** | Bambu Lab | **約 ¥800 〜 ¥1,200** | フィラメント代のみ。圧倒的に安い。 |
| **自家生産 (CF素材)** | Bambu Lab | **約 ¥1,800 〜 ¥2,500** | 材料費は倍になるが、それでも激安。 |
| **海外委託 (MJF)** | **JLCPCB** (中国) 等 | **約 ¥15,000 〜 ¥25,000**<br>($100〜$160 送料込) | 安価だが円安の影響大。「一体型」はかさばるため送料・体積計算が高くなる傾向。 |
| **国内委託 (MJF/SLS)** | **DMM.make** 等 | **約 ¥30,000 〜 ¥50,000** | 品質・サポートは最高だが高額。体積課金のため中身が詰まっていると非常に高価になる。 |

**コスト削減のヒント (外部委託時):**
MJF/SLSなどの粉末造形は「占有体積（Bounding Box）」が価格に大きく影響する。
一体型のV字形状は「隙間（デッドスペース）」が多く、高額になりやすい。
→ **「左右分割型」** に設計変更し、パーツを重ねてコンパクトに配置（ネースティング）することで、コストを **30%〜50% 削減** できる可能性がある。

## 3. 各フェーズにおける推奨戦略

### 3.1 試作フェーズ (Prototyping)
- **推奨**: **PLA (Basic/Matte)**
- **理由**:
    - 印刷が最も安定しており、失敗が少ない。
    - 安価であるため、形状確認のために何度でも出力し直せる。
    - **Tree Hybrid** サポートとの相性が良く、取り外しが容易。

### 3.2 小ロット販売・自家生産 (Small Batch)
- **A案: コスト重視・カラー展開**
    - **素材**: **Matte PLA**
    - **戦略**: 積層痕を「デザインの一部」として許容し、豊富なカラーバリエーションを売りにする。
- **B案: クオリティ重視**
    - **素材**: **CF-PETG / CF-PLA** (カーボン配合)
    - **戦略**: 単価は上がるが、塗装レスで高級感を出し「プレミアムモデル」として差別化する。
    - **必須要件**: 硬化鋼ノズル (Hardened Steel Nozzle)。

### 3.3 ハイエンド・量産 (Mass Production)
- **推奨**: **Nylon (PA12) - MJF/SLS**
- **理由**:
    - 積層痕が一切ないプロ品質。
    - 手汗に強いサラサラした質感。
    - 外部委託（JLCPCB, DMM.make等）となるため、設計の制約（オーバーハング等）から解放される。

## 4. テクニカルノート
- **サポート設定**: 傾斜面や内部空洞が多い本プロジェクトの形状には、Bambu Studioの **"Tree Hybrid"** が最適。
- **PLAの注意点**: 夏場の車内など高温環境（50℃以上）で変形するリスクがあるため、ユーザーへの注意喚起が必要。耐久性を謳うならPETG/ASAへの切り替えを検討すること。
