# プロトタイプ v2 設計仕様書 (Draft)

## 1. 概要
- **目的**: v1の検証結果（保持困難、手首負荷）を解消し、快適な操作性を実現する。
- **主要変更点**:
    - **多面体・屈曲天板構造（Polyhedral/Bent Plate）の採用** [New!]
    - パームレストの一体化（必須アタッチメント化）
    - XY平面への角度付け（ハの字配置）
    - テンティング角の維持・最適化

## 2. 幾何学パラメータ (Geometry)

### 2.1 角度設定
| パラメータ | 設定値 (暫定) | 備考 |
| :--- | :--- | :--- |
| **テンティング角 (Z軸回転)** | 10° 〜 15° | エルゴノミクス調査に基づく推奨値。多面体構造により実現。 |
| **ハの字角度 (XY平面回転)** | **30° (片側 15°)** | **確定**。手首の尺側偏位を防ぐため。 |
| **スロープ傾斜角 (前方)** | **10° 〜 15° (Negative Tilt)** | 手首の背屈を抑制。これ以上の角度は神経圧迫のリスクあり。 |

### 2.2 寸法・配置
- **基本レイアウト**: レバーレス（Hitbox準拠）
- **構造デザイン**: **Polyhedral（多面体）構造**
    - 連続曲面ではなく、平面の組み合わせで構成。スイッチの固定安定性と製造容易性を優先。
    - **Ridge（折れ目）設定**:
        - **位置 (深さ)**: パームレスト手前端から **90mm**。
            - 手首の骨〜手のひらの中心までをカバーし、指の付け根(MP関節)がRidgeに来る想定。
        - **形状**: 鋭利な角を避け、**R2.0mm以上**のフィレット（丸み）を必須とする。
- **左右分割**:
    - [ ] 完全分割型
    - [x] **一体型（V字形状）**
        - **理由**: 膝上プレイ時の安定性を最優先するため（格闘ゲーム用途）。
        - **特徴**: 左右のユニットが中央または底面プレートで接続され、固定された「ハの字」と「テンティング」を持つ。
        - **左右間隔 (Split Distance)**: **50mm** (中央部でのボタン群の距離)
- **ボタン配置・サイズ**:
    - アクションボタン: 24mm
        - **配置**: Ridgeからボタン中心までの距離は **20mm 〜 25mm**。
            - 指を自然に曲げた状態での着地点を狙う。
    - ジャンプ/親指: 30mm
        - **高さオフセット**: 4指の平面より **-5mm 〜 -8mm** 低く配置 (親指の自然な握り込みに対応)。

## 3. 構造・コンポーネント
### 3.1 パームレスト
- **要件**: 天板の傾斜に合わせた**完全一体型**。
    - 水平な机に置くタイプは不可（手首が浮くため）。
- **形状**: 傾斜がついているため、手が滑り落ちないような摩擦係数の高い素材、または「返し」となる形状が必要。

### 3.2 安定性対策
- 底面に滑り止めシート用エリアを確保。
- 内部にウェイト（重り）を入れるスペースを作るか？
- 天板の剛性確保（アクリルなら3mm以上、または支柱追加）。

### 3.3 外側ストッパー（Side Wall）
- **目的**: テンティング傾斜（10-15°）による手の滑り落ち（外側へのスライド）防止。
- **デザイン方針**: **手首側限定＆フェードアウト形状**を採用。
    - **配置**: パームレスト部（小指球付近）にのみ高さを設け、ボタン横（指の可動域）に差し掛かる前に高さをなだらかに消失させる。
        - **理由**: 最外ボタンと縁の距離が **9.5mm** と狭いため、ボタン横に壁があると小指が衝突するリスクが高いため。
    - **寸法**:
        - **高さ**: **5mm 〜 8mm**（最適値）。
    - **形状処理**: 肌への当たりを優しくするため、内側（手が触れる側）にはR3mm以上のフィレット、または外に開くドラフト角を設ける。
