# Project Hanoji 発信方針整理書（Deep Research結果統合版）

## 結論：採用する発信戦略

**「プレイヤー共感型 × 透明性重視 × 二層メディア運用」**

初期3か月間は販売意図を隠し、「自分用に本気で作る制作者」として信頼を積み上げる。
X（日常接触）× note（資産蓄積）の役割分担で、段階的にファン化を図る。

---

## 1. 基本方針（Deep Research統合結果）

### 採用する成功パターン
1. **課題解決の共犯者型**：プレイヤーの悩みを明確化し、解決プロセスを共有
2. **徹底的な透明性型**：コスト・工程・失敗を隠さず開示
3. **備忘録スタンス**：「自己満足＋記録用」で期待値を下げてから価値提供

### 避ける失敗パターン
- カタログスペックの羅列
- 初期からの販売臭
- 失敗の隠蔽
- 専門用語の説明なし

---

## 2. 媒体別運用方針

### X（フロー型：日常接触）
- **頻度**：ほぼ毎日
- **役割**：制作の「今」を高頻度で見せる
- **形式**：写真1枚＋短文、進捗報告、失敗談の即時共有
- **補強要素**：動画や音声はX上での短時間投稿で対応（打鍵音、組み立て過程など）

### note（ストック型：資産蓄積）
- **頻度**：週1本
- **役割**：設計思想・技術解説を体系的に蓄積、体験の詳細レポート
- **形式**：2,000-5,000字の長文、SEO意識したタイトル
- **補強要素**：写真を多用して「体験証明」も兼ねる（組み立て過程、使用感の詳細描写）

---

## 3. 段階別発信設計（12週間）

### 第1フェーズ（1-4週）：「制作者」認知獲得
- **販売臭濃度**：0%
- **目標**：フォロワー50人、「この人は本気で作っている」印象
- **コンテンツ比率**：試作記録40% + 技術解説30% + 設計思想30%

### 第2フェーズ（5-8週）：試行錯誤の共有
- **販売臭濃度**：5%（「欲しいって声が嬉しい」程度）
- **目標**：フォロワー150人、DM/リプでの質問増加
- **コンテンツ比率**：改善プロセス35% + 失敗談25% + 技術解説25% + コミュニティ交流15%
- **体験証明**：Xでの短時間動画（打鍵音、組み立て様子）、noteでの詳細な使用感レポート

### 第3フェーズ（9-12週）：完成度証明＋需要確認
- **販売臭濃度**：10-15%（アンケート実施、コスト計算公開）
- **目標**：フォロワー300人、「販売してほしい」の自然発生
- **コンテンツ比率**：完成品レビュー30% + コスト透明化20% + 販売準備匂わせ15% + その他35%
- **体験証明**：noteでの長期使用レポート、Xでのプレイ実況・勝率報告

## 4. 必須コンテンツ要素（調査結果より）

### 信頼構築要素
- [ ] 「備忘録です」宣言（期待値調整）
- [ ] 費用の完全開示（部品リスト＋価格表）
- [ ] 失敗の「写真＋原因推定＋リカバリ策」セット
- [ ] 先行者・協力者への感謝とクレジット
- [ ] 「知識ゼロ」系の自己開示

### 共感獲得要素
- [ ] 「痛み（体の負担）＋好奇心（技術興味）」の動機構造
- [ ] プレイヤー目線での課題言語化
- [ ] 断定調は「自分の体感」限定
- [ ] 「〜のようだ」「〜かもしれない」の謙虚な推測

### 継続要素
- [ ] 「次にやること」で記事を締める
- [ ] 改善プロセスの継続予告
- [ ] 読者参加型コンテンツ（質問回答、アンケート）

## 6. クロスポスト運用フロー

### note記事公開を軸とした週次サイクル
1. **公開1-2日前**：Xで予告
2. **公開当日**：note投稿 → X要約スレッド
3. **翌日昼**：Xで反応シェア
4. **3日後**：読者質問への回答スレッド
5. **1週間後**：次回予告 + 前回記事再紹介

## 7. KPI設計

### 初期（1-4週）
- Xフォロワー：50人
- Xインプレッション/投稿：500
- note PV/記事：100
- noteスキ数/記事：10

### 中期（5-8週）
- Xフォロワー：150人
- Xエンゲージメント率：3%以上
- DM・リプでの質問：週3件以上
- note記事の保存率：5%以上

### 後期（9-12週）
- Xフォロワー：300人
- **「販売してほしい」系の声：累計10件以上**（最重要KPI）
- アンケート回答数：30以上
- note記事からの外部リンクアクセス：月50クリック

## 8. 販売臭濃度の管理

| 期間 | 濃度 | 具体的表現 |
|---|---|---|
| 1-4週 | 0% | 「自分が使いたくて作っている」のみ |
| 5-8週 | 5% | 「欲しいって言ってくれる人がいて嬉しい」 |
| 9-10週 | 10% | 「もし売るとしたらいくら？」アンケート |
| 11-12週 | 15% | 「量産コスト計算中」「販売方法調査中」 |
| 13週以降 | 20% | BOOTH/BASE公開（投稿の8割は制作コンテンツ維持） |

- [ ] X/noteアカウント開設・プロフィール設定
- [ ] 第1回note記事（動機・設計思想）を公開
- [ ] Xで毎日1投稿を2週間継続してリズム確立

### 継続運用
- [ ] Xでの短時間動画投稿（打鍵音、組み立て様子）を週1-2回

- [ ] 8週目にXアンケートで需要調査
- [ ] 12週目までにBOOTH/BASEの販売ページ準備

### 販売開始判断
- [ ] 「販売してほしい」の声が10件以上自然発生
- [ ] フォロワー300人到達
- [ ] 製品品質が市販品レベルに到達

### YouTube代替手段
- [ ] Xの動画機能で短時間の制作過程・打鍵音を投稿
- [ ] noteで組み立て過程を写真付きで詳細レポート
- [ ] 使用感はnoteでの長文レビュー + Xでのリアルタイム感想で代替

---

## 10. リスク管理

### 避けるべき地雷
- 初期から商品っぽい完成度を見せすぎる（販売臭を疑われる）
- 他者の自作品を批判して自分を優位に見せる
- 専門用語を説明なしで多用する
- 失敗を隠して成功だけアピールする

### 炎上対策
- 断定調は自分の体験のみに限定
- 「個人の感想です」「参考程度に」を適切に使用
- 他者への感謝・クレジットを忘れない
- 謙虚さを常に保つ

---
